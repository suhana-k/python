# 🧠 Quiz Application (Flask Mini Project)

## 📌 Project Description

This is a web-based Quiz Application developed using **Python and Flask**.
It allows users to answer multiple-choice questions and displays the final score at the end and also Shows correct answer and send a invalid message when skip the question.

---

## 🚀 Features

* Multiple-choice quiz questions
* One question displayed at a time
* Timer for each question
* Score calculation at the end
* Shows correct answer and send a invalid message when skip the question
* Simple and user-friendly interface

---

## 🛠️ Technologies Used

* Python 3.x
* Flask
* HTML
* CSS

## ▶️ How to Run the Project

### Step 1: Install Flask

```
pip install flask
```

### Step 2: Run the application

```
python app.py
```

### Step 3: Open in browser

```
http://127.0.0.1:5000/
```
